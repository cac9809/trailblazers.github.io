Project Overview

Hiking is a popular activity and the usage of hiking trails in the Adirondacks has been growing rapidly over recent years. Adirondack Wilderness Advocates is a wilderness advocacy organization that speaks for and protects the wilderness quality of the Adirondack Forest Preserve. There are over 900 different trailheads in the Adirondacks and collecting data from these remote areas to determine trail maintenance can be very time consuming and expensive. To solve this, a series of motion detector sensors are placed around the state park to detect hikers. Activity from the sensors is processed and displayed on a website via an easy to use interactive map of the park.

The implementation uses a cloud based software solution, building on a pre-existing architecture and framework created by previous project teams. This project aims to enhance the current prototype to turn it from a proof-of-concept into a professional software product. The project will take an Agile approach by focusing on regular deliverables and continued sponsor engagement through demos and feedback. A “stretch goal” will be to develop this product to be mobile-friendly as well. The team expects to utilize software engineering techniques/processes such as unit testing, DevOps and CI/CD, and also emphasize secure software development.
